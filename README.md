The consultant will work closely with Data Architecture, Engineering, Payments, Pricing, Channels and Platform teams to ensure identifier changes are fully understood, safely delivered, and traceable across all consuming systems.
Core Focus
Document end‑to‑end client data and identity flows, with SDS as a core master
Analyse and document impact of Single Client ID changes in SDS
Provide clear visibility of dependencies, risks, and downstream impacts
In‑Scope Platforms & Domains
The role will cover documentation and analysis across the following systems and workflows:
SDS (Strategic Data Store) – client data, identifiers, hierarchies
Jetbridge – onboarding, KYC, client origination
CCSS – client and account services supporting Channels and Payments
GTSA / SOLD – global treasury services and accounting identifiers
Customer Systems (CS) – client data consumption and synchronisation
Channels – digital and servicing platforms
BIDG / ETL layers – data movement and reporting dependencies
Barclaycard Issuing – client identity usage in issuing flows
Payments platforms – payment initiation, routing, settlement and reporting workflows
Key Responsibilities
Produce current‑state and target‑state architecture views for client and identity data
Map Single Client ID, CSID, SOLD ID, LEI, BIC and related identifiers across systems
Document system‑to‑system data flows, lineage, and integration patterns
Perform change impact assessments for SDS identifier changes
Identify break points, data quality risks, and regression considerations
Translate complex technical detail into clear, consumable documentation for delivery and governance forums
Architecture and data flow diagrams
Single Client ID impact assessments (per system / per workflow)
Payments and client lifecycle process maps
Clear functional and technical documentation to support safe delivery and testing
Essential Experience
Proven Technical Business Analyst experience in large‑scale banking environments
Strong background in Data Architecture, Master Data, or Client / Party data
Experience documenting complex integration landscapes (APIs, events, batch, ETL)
Solid understanding of Payments and Transaction Banking workflows
Ability to work across multiple platforms and stakeholder groups
